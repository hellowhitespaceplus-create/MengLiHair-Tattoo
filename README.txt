# 纹身与理发工作室（静态多页网站）

## Structure
- index.html (Home)
- tattoo.html
- barber.html
- artists.html
- about.html
- advice.html
- contact.html
- book.html

## Assets
- assets/css/style.css
- assets/js/main.js
- assets/img/*.svg (placeholders)

## 如何运行
Open `index.html` in your browser.
For best results (mobile menu, relative paths), run a simple local server:
- Python: `python -m http.server 8000` then open http://localhost:8000

## 替换图片
Swap the SVG placeholders in `assets/img/` with your real photos (jpg/png/webp),
and keep the same filenames to avoid editing HTML.
